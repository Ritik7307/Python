## Python Tuples
Tuples are ordered collection of data items.They store multiple items in a single variable. Tuple items are separated by commas and enclosed within round brackets ().They are unchangable meaning we can not alter them after creation.

## Manipulating Tuples
Tuples are immutable, hence if you want to add, remove or change tuple items, then first you must convert the tuple to a list. Then perform operation on that lists and convert back to tuple
